# Excel Data Science Salary Dashboard

## Overview

This project presents an interactive Excel dashboard designed to explore global data science salary trends. The dashboard enables users to analyze salaries across job roles, countries, employment types, and work arrangements through dynamic filtering and visualization.

The project demonstrates advanced Excel techniques for data analysis, dashboard development, and business reporting.

---

## Project Objectives

* Analyze salary trends in the data science industry.
* Compare compensation across job roles and locations.
* Explore employment types and work arrangements.
* Create an interactive dashboard for data exploration.

---

## Dataset

The analysis is based on a data science salary dataset containing information such as:

* Job Title
* Salary
* Country
* Employment Type
* Experience Level
* Work Setting
* Platform Information

---

## Dashboard Features

### Salary Analysis

* Average Salary
* Salary Distribution
* Salary Comparison by Role

### Geographic Insights

* Salary by Country
* Regional Trends

### Job Market Analysis

* Job Title Comparison
* Employment Type Analysis
* Work Arrangement Insights

### Interactive Filtering

Users can filter dashboard results by:

* Job Role
* Country
* Employment Type
* Platform
